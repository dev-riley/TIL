#### event

Touchable Component

- TouchableOpacity : 누르면 투명도가 살짝 생기면서 눌렀는지 확인할 수 있다.
- TouchableHighlight : 누르면 배경이 생긴다. 잘 안쓸듯..?
- TouchableWithoutFeedback: 화면의 가장 위에서 일어나는 탭 이벤트를 listen 하는 것. 누르면 화면에는 표시되지 않지만 콘솔창에서 설명이 뜬다.

Pressable: 위의 Touchable Component를 다 포함하고 더 다양한 props를 가지고 있는 event Component이다.
